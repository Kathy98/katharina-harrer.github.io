Website NAKT

WordPress Plugin-Entwicklung, Webentwicklung

Das Projekt

NAKT? Interesse geweckt? NAKT ist der Name einer WordPress-Website, die ich zusammen mit drei Kommiliton*innen im Rahmen eines Studienprojektes im sechsten Semester entwickelt habe. Der Name setzt sich aus den Anfangsbuchstaben unserer Vornamen zusammen.
Das Besondere an der Website war, dass keine vorgefertigten Themes, Plugins, Shortcodes, etc. zum Einsatz kamen, da diese eigenständig zu programmieren waren. Der Schwerpunkt lag dabei auf der Entwicklung mit PHP, wobei auch HTML, JavaScript und CSS zum Einsatz kamen. Zudem habe ich darauf geachtet, dass die Inhalte sowohl in deutscher als auch in englischer Sprache angezeigt werden können, indem ich eine Lokalisierung implementiert habe.

Erlernte Kompetenzen

  • Plugin – Rezepte
    • Shortcode (Formular)
    • Frontend-Formular (Rezepterstellung)
    • User-generated content (Rezepte einreichen über Formular)
    • Erweiterung der Template Hierarchie (Single Post Template)
    • Erweiterung des Customizers
    • Custom Post Type (Rezepte)
    • Einsatz von Medien (Upload eines Rezeptbildes als Anhang)
    • Metadaten (Custom Taxonomy, WP Options)
    • WPDB (Löschen von Taxonomie und Terms)
    • Backend-Erweiterung (Metaboxen, Plugin-Einstellungen, Export)
    • AJAX
  • Widget – Zufällige Beiträge: Konfiguration
  • Sonstiges
    • Content-Selection (WP_Query: Export-Funktion und Widget)
    • JSON als Export-Format (Backend Tipps Plugin)

Meine Aufgaben

Für das Projekt habe ich ein Plugin "Rezepte" entwickelt, um dem*der Nutzer*in die Möglichkeit zu geben, eigene Rezepte anzulegen. Außerdem habe ich ein Widget mit verschiedenen Konfigurationsmöglichkeiten programmiert. Es dient dazu, Beiträge nach dem Zufallsprinzip anzuzeigen. Der*Die Nutzer*in hat ferner die Möglichkeit, Nachhaltigkeitstipps einzureichen, hierfür habe ich einen JSON-Export entwickelt.
Weitere Informationen zum Projekt sowie den kommentierten Quellcode meiner umgesetzten Komponenten findest du auf GitHub.
