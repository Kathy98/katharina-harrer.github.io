Website NAKT

WordPress Plugin-Entwicklung, Webentwicklung

Das Projekt

NAKT? Na, dein Interesse geweckt? Bei NAKT handelt es sich um den Namen der Word­Press Website, die ich innerhalb eines Studienprojektes im sechsten Semester entwickelt habe. Die Besonderheit dabei war, dass keine vorgefertigten Themes, Plugins, Shortcodes, usw. genutzt wurden, da diese eigenständig programmiert werden sollten. Das Hauptaugenmerk lag auf der Entwicklung mittels PHP, wobei auch HTML, JavaScript und CSS zum Einsatz kamen. Zudem wurde auf die Lokalisierung geachtet, um die Inhalte auf Deutsch wie auch Englisch anzeigen zu können.

Erlernte Kompetenzen

  • Plugin – Rezepte
    • Shortcode (Formular)
    • Frontend-Formular (Rezepterstellung)
    • User-generated content (Rezepte einreichen über Formular)
    • Erweiterung der Template Hierarchie (Single Post Template)
    • Erweiterung des Customizers
    • Custom Post Type (Rezepte)
    • Einsatz von Medien (Upload eines Rezeptbildes als Anhang)
    • Metadaten (Custom Taxonomy, WP Options)
    • WPDB (Löschen von Taxonomie und Terms)
    • Backend-Erweiterung (Metaboxen, Plugin-Einstellungen, Export)
    • AJAX
  • Widget – Zufällige Beiträge: Konfiguration
  • Sonstiges
    • Content-Selection (WP_Query: Export-Funktion und Widget)
    • JSON als Export-Format (Backend Tipps Plugin)

Meine Aufgaben

Für das Projekt habe ich ein Plugin "Rezepte" entwickelt, um dem*der Nutzer*in die Möglichkeit zu geben, eigene Rezepte anzulegen. Zusätzlich habe ich ein Widget mit mehreren Konfigurationsmöglichkeiten programmiert. Es ist dafür da, Beiträge per Zufall anzuzeigen. Der*Die Nutzer*in hat ferner die Möglichkeit, Nachhaltigkeitstipps einzureichen, hierfür habe ich einen JSON-Export entwickelt.
Weitere Informationen über das Projekt wie auch den Quellcode kannst du dir auf GitHub ansehen.
