WordPress Plugin-Entwicklung

Im Rahmen eines Hochschulprojekts im sechsten Semester hatte ich die Aufgabe eine Website mittels WordPress zu entwickeln. Die Besonderheit dabei war, dass keine vorgefertigten Themes, Plugins, Shortcodes, usw. genutzt wurden, da diese eigenständig programmiert werden sollten. Das Hauptaugenmerk lag auf der Entwicklung mittels PHP, wobei auch JavaScript, HTML und CSS zum Einsatz kam. Zudem wurde auf die Lokalisierung geachtet, um die Inhalte auf Deutsch wie auch Englisch anzeigen zu können.
Für das Projekt habe ich ein Plugin "Rezepte" entwickelt, um dem Nutzer die Möglichkeit zu geben, eigene Rezepte anzulegen. Zusätzlich habe ich ein Widget mit mehreren Konfigurationsmöglichkeiten programmiert. Es ist dafür da, Beiträge per Zufall anzuzeigen. Der Nutzer hat ferner die Möglichkeit, Nachhaltigkeitstipps einzureichen, hierfür habe ich einen JSON-Export entwickelt.
